# Phase 1: Frontend Visualization Fundamentals

This phase covers the core technologies needed for building genomic visualizations.

## Tutorials

### [1.1 SVG & Canvas Fundamentals](01-svg-canvas/README.md)

Learn the basics of web graphics rendering with SVG and Canvas.

### [1.2 D3.js Core Concepts](02-d3-core/README.md)

Master D3.js selections, scales, axes, and data binding.

### [1.3 Mutation Lollipop Plot ⭐](03-lollipop-plot/README.md)

Build ProteinPaint's signature visualization from scratch.

### [1.4 Genome Browser Track](04-genome-browser/README.md)

Create a mini genome browser with navigation and tracks.

## Prerequisites

- HTML/CSS basics
- JavaScript ES6+ fundamentals
- Basic understanding of genomics (genes, mutations)

## Learning Outcomes

By completing this phase, you will be able to:

- Choose between SVG and Canvas for different use cases
- Create data-driven visualizations with D3.js
- Build interactive genomic visualizations
- Handle coordinate transformations for genomic data
