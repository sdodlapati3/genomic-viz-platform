# Shared Types

TypeScript type definitions shared across tutorials and the capstone project.

## Usage

```typescript
import type { Gene, Variant, Sample } from '@shared/types';
```

## Type Categories

- **Genomic**: Gene, Transcript, Variant, Mutation
- **Clinical**: Sample, Patient, Cohort
- **Visualization**: ChartConfig, ColorScale, Tooltip
- **API**: Request/Response types
